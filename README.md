# Manual de Instalação SCRPAI-IA

Este repositório contém o **manual de instalação** do software SCRPAI-IA desenvolvimento na disciplina Projeto e Desenvolvimento de Software. 

O manual explica passo a passo como preparar o ambiente, configurar chaves API's e executar o software.

## Segurança
- Nunca compartilhe suas chaves API's.  
- Nunca coloque arquivos sensíveis no repositório público.  
- O frontend (`firebaseConfig`) é público por natureza; configure regras do Firebase para proteger os dados.
